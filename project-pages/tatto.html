Tattoo
Our team would like to design a website that can generate tattoos. Users can enter data that they feel are meaningful to them on the web page to generate random tattoos. Those tatto could also be put on body parts via web camera to display the results.
Project Details / Background
ur group has a few passionate tattoo enthusiasts with diverse style preferences, often commemorating special events or days through tattoos. Based on this, we're trying to develop a prototype website focused on generating personalized tattoos.
The website's primary function involves artistic face recognition. Various parameters can be modified by adjusting input values to create unique patterns. Face recognition technology ensures the accurate placement of these patterns.
Although we only tried face generation this time, we may try more other body part recognition in the future, and we learned a lot through this project.
